__Logging__

Visitd can create logs of events that can be useful in debugging. The section controls thee functions:

1.  __Export Logs__
Tap on Export Logs to export the current logs.  

2.  __Enable/Disable Logging__
Tap on Logging Enabled to disabled logging, tap on Logging Disabled to enable logging.

If logging in enabled, tape on Debug Logging to enable or disable debug logging.
