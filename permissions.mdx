__Permissions__

__Location Tracking__

Clicking on the Location Tracking item will launch the System Preferences application and open the settings for Visitd. 

<div style="width: 33.333333%; margin: 1.25rem auto;">
<img src="/home/settings.png" width="200" height="400" />
</div>

Clicking on Location will allow you to set the level of access Visitd has to location. The Location must be set to 'Always' for Visitd to function properly.

<div style="width: 33.333333%; margin: 1.25rem auto;">
<img src="/home/settingsLocationAlways.png" width="200" height="400" />
</div>
